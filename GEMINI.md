# Project Overview

This project is a simple web application for generating text snippets. It displays a list of predefined text options on the left pane. Users can select one or more of these options via checkboxes, and the selected texts are concatenated into a text area on the right pane. The user can then copy the generated text to the clipboard.

The application is built with plain HTML, CSS, and JavaScript, with no external libraries or frameworks.

# Running the Application

To run the application, simply open the `index.html` file in a web browser.

# Key Files

*   `index.html`: The main HTML file that defines the structure of the web page.
*   `style.css`: The stylesheet for the application, defining the layout and appearance of the elements.
*   `script.js`: The JavaScript file that contains the application's logic. It populates the text options, handles user interactions (checkbox clicks), and implements the copy-to-clipboard functionality.
*   `teksten.md`: A Markdown file containing the text snippets used in the application. Note that these texts are also hardcoded in `script.js`.
*   `CARDION_logo.png`: An image file for the project's logo, which is not currently used in the application.

# Development Conventions

The code is written in a straightforward manner with vanilla JavaScript, HTML, and CSS. There are no specific build tools or development environments required. The JavaScript code is self-contained and manipulates the DOM directly. The styling is simple and uses Flexbox for layout.
